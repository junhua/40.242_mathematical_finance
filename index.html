Question 4. (6 points)¶

Assume no short selling. Find the minimum variance portfolio of the four stocks. What is the corresponding mean and standard deviation of the return of this portfolio? Find the efficient portfolio with $r_p = 1.6\%$. What is the corresponding standard deviation of the return of this portfolio?

Solution

Setup the optimization problem as follows:

Objective: minimze $\sum_{i,j}^n w_iw_j\sigma_{i,j}$

Subject to:

  1. $\sum_i^n w_i = 1$
  2. $r_p > 1.6\%$ (Only applicable to the efficient portfolio with $r_p = 1.6\%$)

Solving the optimization problem, we get the answer as follows:

## objective function: minimize portfolio std

def show_result(title, x):
    num_bins = 50
    n, bins, patches = plt.hist(x, num_bins, normed=1, facecolor='blue', alpha=0.5)

    # add a 'best fit' line y
    y = mlab.normpdf(bins, 0, 1)
    plt.plot(bins, y,'r')
    plt.title(title)
    plt.show()
    print "sample average: %s, standard deviation: %s"%(round(np.mean(x),3),round(np.std(x, ddof =1),3))
    print "skewness: %s, and kurtosis: %s"%(round(skew(x),3), round(kurtosis(x)+3,3))

Method 1: CLT Approximation¶

If $U_1, U_2, ... U_n$ are iid as U(0,1), then $Z = \frac{\sum{U_i}-{n}/{2}}{\sqrt{{n}/{12}}}$ has an approximate N(0,1) distribution.

Method 2: Box-Muller Method¶

Generate two independent random number $U_1, U_2$ from U(0,1) distribution and return $Z_1 = \sqrt{-2lnU_1} Cos(2\pi U_2)$, $Z_2 = \sqrt{-2lnU_1} Sin(2\pi U_2)$

def bm(n=1000):
    r = np.sqrt(np.array(np.random.exponential(2, size=n)))
    theta =np.array(np.random.uniform(0, 2 * np.pi, size = n))
    return r * np.cos(theta)

Method 3: Polar Method¶

1.Generate two randum numbers $U_1$ and $U_2$ from U(0,1) distribution.

2.Set $V_1 = 2U_1-1$, $V_2=2U_2-1$ and $S = V_1^2 + V_2^2$

if S>1, go to step 1, else:

$Z_1 = \sqrt{\frac{(-2)\log(S)}{S}}*V_1$ and $Z_2 = \sqrt{\frac{(-2)\log(S)}{S}}*V_2$
